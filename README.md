# 开发目的
现在矿池基本上都是 0.1ETH起付 但是作为个人一两张显卡需要挖一个月甚至更久才能挖出0.1ETH，
所以集合了一堆散户挖ETH组成一支"矿车"车队，这样就可以2周或者更快就能起付。
但是人力计算每人收益难免会出差错，故写出一个数据版来参考数据。
